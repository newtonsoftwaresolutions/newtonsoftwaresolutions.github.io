v3.0.0 - Released 05/03/2010

  • Complete rewrite of the entire application for the .NET framework
  • Lots of enhancements in functionality such as :
    • Much quicker and less resource intensive than previous versions.
    • Compatible with Windows 7, Windows Vista and Windows XP. Extensively tested on all these environments.
    • Optimised to run very quickly with minimum system resources used, using the latest microsoft .NET framework
    • Can be setup to backup files "realtime", i.e. as soon as files are changed in a source folder they get immediately copied to the destination folder
    • System tray messages are displayed to let you know what clone has done
    • Statistics tell you how many files have been analysed / copied
    • Much improved delete functionality.
    • Automated detection of new versions of clone. Upgrades happen at the click of the button and take less than 5 seconds.

v2.1.0 - Released 03/07/2003

  • 2 way sync problem fixed.
  • Several minor bugs fixed.

v2.0.1 - Released 14/02/2003

  • Fixed problem with remove set option.
  • Destination disk full error handled gracefully.
  • Advanced screen resized correctly so buttons are visible.

v2.0.0 - Released 31/01/2003

  • Synchronisation method included
    • 2 way sync, source <--> destination
    • 1 way sync, source --> destination
    • 1 way sync, destination --> source
  • Option to delete removed files in destination directory.
  • Option to start Clone with windows, can delay start by "x" minutes as well.
  • File filter option.
  • User interface improved on all screens, improved features include :-
    • Hot Keys, e.g. ALT+F etc.
    • Tab sequences enabled.
    • Colours and Bold fonts used.
    • Toolbars rather than buttons used.
    • Context sensitive help (whatsthis help).
  • Improved information on main screen, i.e. next run time last run time and countdown to next run.
  • Advanced settings hidden from main options.
  • Improved calculation for status bar.
  • Help file improved including context sensitive help.
  • Registration method included.
