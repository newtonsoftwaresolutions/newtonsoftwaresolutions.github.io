Getting Started

The clone user interface is based on a tabbed layout. All screens can be easily navigated to at the click of a button. You can get help on any screen of clone by pressing the F1 key (top left of keyboard). The tabs are shown below



The first thing you need to do is configure the files that you want backed up. This is done on the following screen which is accessed from the "Sets" tab



For detailed information on the operation of this tab see :- Sets Tab
The screenshot above shows the various sets configured, e.g. "external hd backup". The list will be empty the 1st time you run clone. Once you add a new set using the New button you will see it in the list. The check box to the right of the set name allows you to include (when checked) or exclude the set from a backup. For example if you configure Clone on your laptop you will want to exclude files that are normally backed up to your work server when you are using your laptop at home.
The New / Properties button takes you to another configuration screen. This is shown below :-



This screen enables you to configure the exact details of the current set. You can choose to backup an entire directory, a single file or files that match a filter. You can set several options relating to a set, these are described in more detail in Set Configuration Screen .
