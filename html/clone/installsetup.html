Installation

Clone can be downloaded from the following web site :-

http://newtonsoftware.net/clone/CloneSetup.msi

Clone is installed by double clikcing CloneSetup.msi file. This is a windows installation file. Once you have downloaded the installation file just double click on it. The default installation folder for Clone is :- 

c:\program files\clone3
The icon group created by the installation looks something like :-



These icons are explained below :-
  • Clone : Runs Clone
  • Clone Help : View Clone help file.

An icon should also have been created on your desktop.

Running Clone

To run clone click on the the Clone Icon.
