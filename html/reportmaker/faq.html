How do I access the application?

Click on https://apps.newtonsoftware.co.uk/reportmaker

Are there instructions for Report Maker?

View the online User Guide or download the MS word User Guide or the pdf User Guide

What’s the current version of Report Maker?

An online version of Report Maker has been released in 2012

What new features are there in the new release

Report Maker Online incorporates all the features of the previous version of Report Maker, but does it all through a web browser. You statements, classes and reports can all be stored securely online, for recall at any point in the future

How do I know my data is secure?

Report Maker uses https security. This ensures that all the data transfered between the server and your computer is encrypted and no one can access it. Data on the server is backed up daily so you can rest assured that your statements and reports are safe. All users of the system have to be logged on. Only data that the users own can be viewed by them, with the exception of statements which can be shared between users.

How can I keep informed of new changes to Report Maker?

Drop me an email and I’ll add you to my distribution list

How much does Report Maker Cost?

Currently Report Maker Online is free to use. There is likely to be a small charge in the future.

Where can I get statements from?

There are many statements included when you login to the application. These are categorised by subject area and Key Stage.

How can I get support for Report Maker?

Email based support will be provided, where ever possible this will be done on the same day the email is received. If you require telephone support for an additional cost, please contact email newtonsoftware for details
